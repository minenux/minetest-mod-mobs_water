# minetest-mod-mobs_fish

fish for mobs_water

## Requirements

* Minetest >= 0.4.17
* default
* mobs

## Settings

### mobs_fish.fish_sprite

Enable usage of sprites event mesh of texture map

Enable use sprites on slow cpus that dont have good 3d opengl procesing

default: **false**
